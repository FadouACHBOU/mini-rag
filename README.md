# Mini-RAG
This is a minimal implementation of a rag model for question answering
# Requirements 
python 3.13 or later 
### Install Python using miniconda

1) Download and install miniconda from here : [here](https://anaconda.com/api/installers/Miniconda3-latest-Windows-x86_64.exe)
2) create a new environement using the followning command : 
```bash
$ conda create -n mini-rag-app python=3.13
```
3) Activate the environment
```bash
$ conda activate mini-rag-app
```

### (Optional) Setup you command line interface for better readability
```bash
export PS1="\[\033[01;32m\]\u@\h:\w\n\[\033[00m\]\$ "
```
